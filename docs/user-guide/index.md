# User Guide

This guide provides comprehensive instructions for technical writers, developers, and product managers using SudoDocs to automate documentation workflows.

## In This Section

* [Compose Release Notes](release-composer.md): Generate consolidated changelogs from Jira or CSVs.
* [Write a Single Release Note Entry](single-note.md): Create a polished entry for a specific ticket: feature, enhancement, bug fix, or a known issue.
* [Author a Feature](feature-author.md): Draft comprehensive feature documentation.
* [Check API Readiness](api-readiness.md): Validate OpenAPI specifications and upload to ReadMe.
* [Universal Doc Converter](api-readiness.md): Translate documentation between different markup languages.
